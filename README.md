# Calendar Booking App

Небольшое веб-приложение для бронирования временных слотов в календаре.

Проект учебный: основная цель — пройти полный цикл разработки с использованием AI-агентов в подходе **Design First**. Сначала фиксируется API-контракт, затем по нему независимо реализуются фронтенд, бэкенд, тесты и деплой.

## Что делает приложение

В приложении есть две условные роли:

- **Владелец календаря** (один предустановленный профиль): создаёт типы событий, просматривает все бронирования.
- **Гость** (без регистрации): выбирает тип события, выбирает свободный слот в окне 14 дней, создаёт бронирование.

## API-first / Design First

API-контракт является источником истины для фронтенда и бэкенда.
